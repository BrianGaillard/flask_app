About This Project

This project uses what we have learned from the AI 4 All Open Learning course as well as our past experience with programming to create a functional web application. We used the Flask web framework to create this app as well as Heroku to host the application.

We use data that is publicly acessible using the financial statements found on the East Orange website. Using these documents, we apply multivariate linear regression to predict possible future financial values.
