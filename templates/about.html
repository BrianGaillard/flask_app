About This Project

This project uses what we have learned from the AI 4 All Open Learning course as well as our past experience with programming to create a functional web application. We used the Flask web framework to create this app as well as Heroku to host the application.

We use data that is publicly acessible using the financial statements found on the East Orange website. Using these documents, we apply univariate linear regression to predict possible future financial values.

What is AI?

AI stands for "Artificial Intelligence" and refers to technology that allows machines to simulate human qualities. This can include recommending videos you may like or driving your car around for you. If you have ever used Google maps or talked to Siri, you have interacted with AI. AI is all around us, but we don't always realize it. Its uses are many and it is an incredible area of computer science. Machine learning is a sort of subset of AI that focuses on giving computers the ability to make predictions and decisions based on data, and this is what we made use of for this application.
